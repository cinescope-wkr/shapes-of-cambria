# Model / Code

Put the project’s model code and reproduction instructions here.

Suggested structure:

- `model/src/` (library code)
- `model/train.py` / `model/infer.py` (entry points)
- `configs/` (experiment configs)
- `scripts/` (download / preprocessing / eval helpers)

Keep large files out of git (datasets, checkpoints, long videos); use GitHub Releases or external storage instead.
